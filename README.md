# IncidentDesk

A full-stack incident & request management tracker with role-based access control.

## Tech Stack

**Frontend:** React · TypeScript · Vite · Tailwind CSS · TanStack Query · React Hook Form + Zod
**Backend:** Node · Express · TypeScript · Prisma · PostgreSQL · JWT
**Tooling:** Vitest · GitHub Actions
